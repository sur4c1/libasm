# libasm

## Usage

### Compile the library

```sh
make libasm
```

### Compile the testers

```sh
make testers
```

### Compile the tester for a specific function

```sh
make [function_name]
```

Note: The function name does not include the `ft_` prefix.

### Lauch the tests

```sh
./test.sh
```
